# Compliance Evidence Collection

## Overview

This repository contains my Week 4 internship assignment which was completed during the Sohail Smart Solutions Summer Training Program 2026.

The objective of this assignment was to be able showcase and demonstrate how the organizations are able to provide objective audit evidence to verify that information security controls are operating effectively. This work is able to build upon the previous ISO/IEC 27001:2022 Gap Assessment and Security Policy Development assignments completed during Week 4.

The assessment uses the fictional organization **EduTrack Learning Solutions**, where the selected Annex A controls are mapped to the evidence required during an ISO/IEC 27001 audit.

## Repository Contents

- `evidence-register.md` – Summary of the Evidence Register.
- `evidence-register.csv` – Evidence Register in CSV format.

## Assessment Scope

The repository documents:

- Selected ISO/IEC 27001:2022 Annex A controls
- Required audit evidence
- Evidence owner
- Collection frequency
- Current implementation status

## Related Frameworks

- ISO/IEC 27001:2022
- ISO/IEC 27001 Annex A
- Information Security Management System (ISMS)
- Governance, Risk and Compliance (GRC)
