# Compliance Evidence Register

## Overview

This Evidence Register is able to demonstrate how EduTrack Learning Solutions can provide objective evidence that the selected ISO/IEC 27001:2022 Annex A controls are operating effectively. The register links each control with the evidence an auditor would expect to review, the responsible owner,along with the collection frequency, and even the current implementation status.

## Evidence Register

| Annex A Control | Required Audit Evidence | Evidence Owner | Collection Frequency | Status |
|-----------------|-------------------------|----------------|----------------------|--------|
| A.5.1 – Policies for Information Security | Approved Information Security Policy, version history, annual review records | Information Security Manager | Annual | Active |
| A.5.9 – Inventory of Information Assets | Asset inventory reports, ownership records, inventory review reports | IT Asset Manager | Quarterly | Active |
| A.5.23 – Information Security for Cloud Services | Cloud configuration reports, supplier security assessments, baseline configuration records | Cloud Administrator | Monthly | In Progress |
| A.6.3 – Information Security Awareness | Training attendance records, completion certificates, learning management reports | Human Resources Manager | Annual | Active |
| A.6.8 – Information Security Event Reporting | Incident tickets, investigation reports, helpdesk records | Security Operations Team | Ongoing | Active |
| A.8.2 – Privileged Access Rights | Privileged access review reports, approval records, administrator account register | IT Security Manager | Quarterly | In Progress |
| A.8.5 – Secure Authentication | MFA configuration reports, password policy, authentication settings | System Administrator | Quarterly | Active |
| A.8.15 – Logging | SIEM dashboards, audit log exports, monitoring summaries | Security Operations Team | Monthly | Active |

## Key Findings

- Governance controls tend to require periodic review and documented approval.
- Asset inventories should remain accurate through the scheduled validation.
- Cloud security evidence should be collected consistently throughout the year.
- Privileged access reviews is able to provide assurance that elevated permissions remain appropriate.
- Security logging is able to support monitoring, investigations, and even audit activities.
- Continuous evidence collection is able to improve audit readiness and even reduce the preparation effort.

## Conclusion

Maintaining a structured Evidence Register is enables EduTrack Learning Solutions to be able demonstrate that the information security controls are operating effectively rather than existing only as documented policies. Regular evidence collection is able to strengthen the governance, as well as support continual improvement, and even helps the organization prepare efficiently for future ISO/IEC 27001 internal and external audits.
